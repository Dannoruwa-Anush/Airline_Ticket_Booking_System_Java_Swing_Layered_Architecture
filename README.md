# Java Swing Application

A desktop application built using Java Swing, following a Layered Architecture design with Singleton and Factory Design Pattern . 

## Project Structure 
- Entity layer
- Data Access Layer (DAO) layer
- Service Layer
- Controoler layer
- Presentation layer
